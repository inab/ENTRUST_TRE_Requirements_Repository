
# Code of Conduct

## Our Commitment

We, as contributors and maintainers of the EOSC-ENTRUST Requirements Repository, are committed to providing a welcoming, inclusive, and professional environment for collaboration.

Participation in this repository is governed by mutual respect, constructive dialogue, and a shared commitment to quality, transparency, and integrity.


## Expected Behaviour

All participants are expected to:

- Engage respectfully and professionally with others
- Be considerate of differing viewpoints and experiences
- Provide constructive feedback and accept it gracefully
- Focus discussions on technical, methodological, or governance-related matters
- Respect project governance decisions and review processes


## Unacceptable Behaviour

Unacceptable behaviour includes, but is not limited to:

- Harassment, discrimination, or intimidation of any kind
- Offensive, derogatory, or inflammatory language
- Personal attacks or repeated disruptive behaviour
- Publishing private information without consent
- Any conduct that undermines constructive collaboration


## Scope

This Code of Conduct applies to all interactions related to this repository, including:
- Issue discussions
- Pull request reviews
- Documentation contributions
- Any communication channels linked to the repository

---

## Enforcement Responsibilities

The repository maintainers and designated EOSC-ENTRUST governance representatives are responsible for enforcing this Code of Conduct.

They will:
- Assess reported incidents fairly and confidentially
- Take appropriate action to address violations
- Act in alignment with the EOSC-ENTRUST governance framework

---

## Enforcement Actions

In response to unacceptable behaviour, maintainers may take actions including:
- Issuing a warning
- Requesting corrective behaviour
- Temporarily restricting participation
- Removing content that violates this Code of Conduct
- Permanently banning a contributor in severe cases

Actions will be proportional to the nature and severity of the violation.

---

## Reporting Issues

If you experience or witness behaviour that violates this Code of Conduct, please report it to the repository maintainers or the EOSC-ENTRUST Change Control Authority.

Reports will be handled with discretion and respect for confidentiality.
